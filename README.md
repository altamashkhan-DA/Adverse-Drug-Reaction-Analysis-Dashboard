 #  Adverse Drug Reaction Analysis Dashboard

##  Project Overview

This project presents an interactive **Power BI dashboard** designed to analyze **Adverse Drug Reactions (ADR)** using patient demographic information, drug details, and reaction severity.

The dashboard helps identify reporting patterns across age groups, gender, severity levels, and medications, enabling better healthcare decision-making.

---

##  Objectives

* Analyze adverse drug reaction reports.
* Identify drugs associated with severe reactions.
* Compare reaction severity across demographic groups.
* Provide interactive filtering for better exploration.
* Generate actionable business insights.

---

##  Dashboard Features

### Executive Dashboard

* Total Adverse Events
* Severe Reactions
* Severe Reaction Rate
* Drugs Monitored
* Drug-wise Severe Reactions
* Severity Distribution
* Cases by Age Group
* Dynamic Insights

### Demographic Dashboard

* Age Group Distribution
* Average Patient Age by Severity
* Severity by Age Group
* Gender Distribution
* Severity Distribution by Gender
* Key Demographic Insights

---

##  Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

---

##  Key Insights

* Adults account for the highest number of reported adverse drug events.
* Severe reactions represent a small percentage of total reported cases.
* Mild reactions are the most frequently reported severity level.
* Gender distribution of reported cases is nearly balanced.
* Patients with severe reactions have a slightly higher average age.
